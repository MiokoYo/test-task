# Входные данные для проверки формы возможности оформления паспорта

## Граничные значения:
 
 Входные данные
 1. 0 - ошибка
 2. -1 - Ошибка
 3. 1 - Нельзя оформить
 4. 13 - Нельзя оформить
 5. 14 - Можно оформить
 6. 115 - Можно оформить
 7. 116 - Ошибка

 Класс, дающий результат "Ошибка"

 Отрицательные значения
